The editor can tag a list with a named style. It serializes to a `:::list{.class}`
directive and the site renders it as `<ul class="class">`.

Růžové tečky:

:::list{.dots-pink}
- jablko
- hruška
- švestka
:::

Fajfky:

:::list{.checkmarks}
- hotovo
- zkontrolováno
- nasazeno
:::

Plain prose with stray colons stays intact: klíč:hodnota, sraz v 8:30, Pozor:důležité.

The editor's YouTube button emits a `::youtube{#id}` directive, which the site renders
as a styled `<iframe class="youtube-embed">`:
